# Development

## Requirements
* [Node.js (20+)](https://nodejs.org/en)
* [npm](https://www.npmjs.com/)
* [Typescript](https://www.typescriptlang.org/)
* [Docker](https://docs.docker.com/engine/)

## Cli

* Run needed services (postgres)
```bash
docker compose up -d
```

* Install dependencies
```bash
npm i
```
